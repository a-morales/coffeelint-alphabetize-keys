# coffeelint-alphabetize-keys

## Examples
```coffee
{keyA, keyB, keyC} # Good
{keyC, keyB, keyA} # Bad
```

## Installation

```
npm install coffeelint-alphabetize-keys
```

## Usage

Put this in your coffeelint config:

```json
"alphabetize_keys": {
  "module": "coffeelint-alphabetize-keys",
}
```
